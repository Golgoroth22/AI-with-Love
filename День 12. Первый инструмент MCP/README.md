# День 12. Первый инструмент MCP

1. Реализуйте простой инструмент (например, яндекс трекер) и зарегистрируйте его в MCP
2. Подключите агента, чтобы он мог вызвать этот инструмент через MCP
3. Получите результат (например количество открытых тасок)

- `Результат`: Агент делает вызов к вашему инструменту и получает результат
- `Формат`: Видео + Код
